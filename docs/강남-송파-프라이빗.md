- 작성일: 2026-05-08
- 검색 범위: 강남, 역삼, 논현, 선릉, 신논현, 언주, 선정릉, 삼성, 봉은사, 송파(잠실·석촌·방이·문정·송리단길)
- 목적: **프라이빗한 이자카야 및 오마카세**를 실제 예약 가능성 중심으로 추려서 정리

## 1. 요청 조건 요약
- 요청 핵심은 **프라이빗한 공간**이다.
- 단순히 분위기 좋은 곳이 아니라, 아래 중 하나가 확인되는 곳을 우선했다.
  - 개별룸
  - 반개별룸 / 독립공간
  - 룸술집 / 대관 가능
  - 카운터 소수좌석 / 예약제 소규모 운영
- 음식은 이자카야와 오마카세를 중심으로 보되, **실전적으로 비슷한 프라이버시를 주는 일본식 다이닝**도 함께 봤다.

## 2. 검색 기준과 검증 원칙
- **공식 사이트 / 예약 페이지**를 최우선으로 봤다.
- 그다음 **예약 플랫폼 / 지도 / 리뷰 / 블로그 후기**를 참고했다.
- 아래 기준으로 판정했다.
  - **확인됨**: 룸, 정원, 대관, 좌석 구조가 페이지에 직접 적혀 있음
  - **추정됨**: 후기나 예약 맥락상 프라이빗 성격이 강하지만 룸 문구는 약함
  - **미확인**: 프라이버시는 좋아 보이지만 직접 근거가 부족함
- 오픈 홀, 바석 위주, 프라이버시 근거가 약한 곳은 제외했다.
- 강남/송파권을 기준으로 하되, 실전 대안으로 압구정·가로수길 등 인접권 일부도 포함했다.
