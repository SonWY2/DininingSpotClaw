# 강남·송파권 프라이빗 데이트 식당 리포트

## 1. 요청 조건 요약
- 지역: 강남, 역삼, 선릉, 삼성, 잠실새내, 송파
- 목적: 2인이 조용하게 데이트할 수 있는 식당 탐색
- 핵심 조건:
  - 오픈된 홀, 바 좌석, 시선이 트인 공간은 제외
  - 커튼석, 개별룸, 프라이빗 룸처럼 **남들 시선에 구애받지 않는 공간** 우선
  - 음식 종류는 한정하지 않음
- 추가 요청 반영:
  - **일식/이자카야 위주 후보를 별도 확장 검색**
  - 기존 보고서에 없던 후보만 우선 추가

## 2. 검색 기준과 검증 원칙
- 공식/예약/리뷰 플랫폼에서 **룸, 개별룸, 커튼석, 프라이빗 룸** 관련 표현이 확인된 곳만 우선 반영
- 단순히 분위기가 좋거나 조용하다는 이유만으로는 포함하지 않음
- 2인 룸 가능 여부가 불명확하면 **조건 확인 필요**로 분류
- 일식/이자카야 추가 검색에서는 아래 키워드를 중점 반영
  - 강남/역삼/선릉/삼성/잠실새내/송파 + 이자카야 룸
  - 프라이빗 룸, 개별룸, 커튼석, 룸술집, 예약 가능
- 본 리포트는 현재까지 수집된 정보 기준이며, 실제 예약 전에는 아래를 최종 확인해야 함
  - 2인도 룸 배정 가능한지
  - 커튼석/분리석이 예약 가능한지
  - 룸차지, 코스 주문, 보틀 주문 등의 조건이 있는지
  - 다찌/바 좌석이 아닌 **독립 공간**으로 배정 가능한지

## 3. 추천 후보 상세

## 3-1. 강추천

### 고우가 강남 역삼역점
- 위치: 역삼
- 예상 가격대: 고가
- 인원 적합성: 2인 데이트 적합
- 공간 유형: 전 좌석 프라이빗 룸
- 콜키지: 확인 필요
- 추천 이유: 전체 좌석이 프라이빗 룸이라는 근거가 비교적 명확해, 이번 요청 조건과 가장 잘 맞는 축에 속함.
- 유의사항: 캐주얼 데이트보다는 포멀하고 특별한 날 식사에 더 가까울 수 있음.
- 근거 수준: 높음

### 사색 고급요리집
- 위치: 역삼
- 예상 가격대: 고가
- 인원 적합성: 2인 데이트 적합
- 공간 유형: 전 좌석 프라이빗 룸 / 2인실 언급
- 콜키지: 확인 필요
- 추천 이유: 2인실 관련 근거가 있어, 남들 시선에서 벗어난 식사를 원하는 경우 우선순위가 높음.
- 유의사항: 분위기가 차분하고 정갈해 다소 격식 있게 느껴질 수 있음.
- 근거 수준: 높음

### 민규 삼성점
- 위치: 삼성동
- 예상 가격대: 고가
- 인원 적합성: 2인 데이트 적합
- 공간 유형: 전 좌석 프라이빗 룸
- 콜키지: 확인 필요
- 추천 이유: 삼성권 후보 중 프라이빗 룸 근거가 가장 뚜렷한 편이라 안정감이 있음.
- 유의사항: 가격대가 높고, 한우 오마카세 성격이 강함.
- 근거 수준: 높음

### 머무를 정 세이로무시 선릉점
- 위치: 선릉
- 예상 가격대: 중상~고가
- 인원 적합성: 2인 데이트 적합
- 공간 유형: 개별 룸 위주
- 콜키지: 확인 필요
- 추천 이유: 차분한 분위기와 개별 룸 중심 구성이라는 근거가 반복적으로 확인됨.
- 유의사항: 로맨틱한 연출보다는 조용하고 단정한 식사 쪽에 가까움.
- 근거 수준: 중상

### 토우지
- 위치: 잠실새내
- 예상 가격대: 중상
- 인원 적합성: 2인 데이트 적합
- 공간 유형: 개별룸 / 커튼 친 프라이빗 좌석
- 콜키지: 확인 필요
- 추천 이유: 잠실새내권에서 이번 조건에 가장 직접적으로 맞는 편. 커튼/분리 좌석 근거가 있어 오픈홀 회피 니즈와 잘 맞음.
- 유의사항: 업장 성격상 늦은 시간에는 소음 편차가 있을 수 있음.
- 근거 수준: 중상

### 동화고옥 롯데월드몰점
- 위치: 잠실
- 예상 가격대: 중상~고가
- 인원 적합성: 2인 데이트 적합
- 공간 유형: 전 좌석 룸 계열
- 콜키지: 확인 필요
- 추천 이유: 잠실권에서 룸 구조 근거가 비교적 뚜렷해 시선 차단 관점에서 안정적임.
- 유의사항: 데이트 감성보다는 정갈하고 격식 있는 식사 쪽으로 느껴질 수 있음.
- 근거 수준: 중상

## 3-3. 일식·이자카야 추가 후보

> 아래 후보들은 추가 검색으로 발굴한 일본식 중심 후보입니다. 다만 상당수는 리뷰/플랫폼 태그에서 룸 또는 분리 좌석이 확인된 수준이라, **실제 예약 시 2인 룸/세미룸 가능 여부를 꼭 다시 확인**하는 것이 좋습니다.

- 위치: 역삼
- 예상 가격대: 중상

- 공간 유형: 룸 있음 / 예약 가능
- 콜키지: 확인 필요
- 추천 이유: 예약 플랫폼에서 룸 시설이 직접 확인된 편이라, 일식 이자카야 계열 추가 후보 중에서는 실전성이 높은 편.
- 유의사항: 2인 전용 룸인지, 다인용 룸만 운영하는지는 확인 필요.
- 근거 수준: 중상

- 공간 유형: 룸 / 분리석 / 층별 좌석 분리 가능성

- 유의사항: 좌석 타입이 섞여 있을 가능성이 있어 예약 시 가장 안쪽 또는 룸 좌석 요청이 필요.

- 근거 수준: 중하
