# 마츠야마 로컬 맛집 조사 보고서

## 1. 요청 조건 요약
- 지역: 일본 에히메현 마츠야마(松山)
- 인원: 여자 3명
- 목표: 로컬 느낌이 강한 맛집
- 제외: 한국인 관광객 위주, 관광지형 맛집
- 참고 조건: Google Maps 평점 4.3+, 리뷰 50+ 선호

## 2. 검색 기준과 검증 원칙
- 공식 사이트 > 예약/리스트 페이지 > 지도/리뷰 페이지 순으로 확인
- 관광객용 대표 맛집보다, 현지인 분위기·조용한 분위기·식사 완성도 우선
- 현재 보고서는 **지금까지 조사된 정보만**으로 작성
- Google Maps 평점/리뷰 수는 일부 후보에서 최종 확정 전

## 3. 추천 후보 상세

### 화식 미요시 (和食みよし)
- 위치: 마츠야마시 센주마치 3-4-2
- 예상 가격대: 런치 1,000엔대 / 디너 5,500~7,000엔대
- 인원 적합성: 3명에 적합
- 공간 유형: 카운터 중심의 소규모 숨은 맛집(총 12석)
- 대표 메뉴: 모둠 회, 옥수수 튀김, 점심 정식(제철 식재료 구성), 가이세키 코스, 일본주
- 추천 이유: 로컬 감성이 가장 강하고, 조용하고 단정한 분위기
- 유의사항: 좌석 수가 적어 혼잡 시 불리
- 근거 수준: 높음

### 잇핀 오료리 오오타키 (一品 御料理 おゝたき)
- 위치: 마츠야마시 니반초 1-3-5
- 예상 가격대: 디너 12,000엔대
- 인원 적합성: 3명 가능, 다만 고급 식사 성격이 강함
- 공간 유형: 카운터 + 개인실
- 대표 메뉴: 가이세키 코스, 스키야키 코스, 제철 전골 코스, 일품 요리, 오뎅, 교토식 식재료 요리
- 추천 이유: 실력 있는 일본요리집, 조용한 식사에 적합
- 유의사항: 예산이 높은 편
- 근거 수준: 중상

### 타이야/다이야 (鯛や)
- 위치: 마츠야마시 미쓰 1-3-21
- 예상 가격대: 2,000엔대
- 인원 적합성: 3명 가능
- 공간 유형: 등록유형문화재 고택, 좌식/테이블/개인실
- 대표 메뉴: 도미밥 정식(타이메시 젠), 도미 회, 도미 국물, 자코텐, 시마히지키, 제철 반찬
- 추천 이유: 분위기와 로컬성 모두 좋음
- 유의사항: 중심가에서 다소 떨어짐
- 근거 수준: 중상

### 원조 우와지마 도미밥 마루스이 본점 (元祖 宇和島鯛めし 丸水 本店)
- 위치: 마츠야마시 오카이도 3-7-8
- 예상 가격대: 2,000엔~
- 인원 적합성: 3명 가능
- 공간 유형: 42석, 칸막이형 개인실 있음
- 대표 메뉴: 우와지마 도미밥, 도미 튀김덮밥, 자코카츠, 어린이 세트
- 추천 이유: 대표 향토요리를 안정적으로 먹기 좋음
- 유의사항: 관광 접근성이 높아 ‘숨은 맛집’ 느낌은 약함
- 근거 수준: 높음

### 향토요리 고시키 (郷土料理 五志喜 本店)
- 위치: 마츠야마시 산반초 3-5-4
- 예상 가격대: 런치 1,000~3,000엔 / 디너 3,000~5,000엔
- 인원 적합성: 매우 좋음
- 공간 유형: 대형 매장, 단체/개인실 대응
- 대표 메뉴: 마쓰야마 도미밥, 우와지마 도미밥, 도미소면, 오색소면, 자코텐, 자코카츠, 미캉비어, 도고 지비어
- 추천 이유: 에히메 향토요리를 한 번에 보기 좋음
- 유의사항: 관광객 비중이 높아 로컬 숨은 맛집 느낌은 약함
- 근거 수준: 높음

### 마쓰야마 도미밥 아키요시 본점 (松山鯛めし 秋嘉 本店)
- 위치: 마츠야마시 오카이도 3-5-1
- 예상 가격대: 2,000~2,999엔
- 인원 적합성: 3명 가능
- 공간 유형: 44석, 개인실 있음
- 대표 메뉴: 마쓰야마 도미밥, 우와지마 도미밥, 도미소면, 자코텐 3종 모둠, 자코카츠, 도미밥 정식
- 추천 이유: 접근성 좋고 메뉴 구성이 무난함
- 유의사항: 상권 중심이라 관광형 느낌이 있음
- 근거 수준: 중상
