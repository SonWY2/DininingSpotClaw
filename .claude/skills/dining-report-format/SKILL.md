# Dining Report Format

## Instructions
Use this structure:

1. 요청 조건 요약
2. 검색 기준과 검증 원칙
3. 추천 후보 상세
4. 비교 표
5. 최종 추천

## Candidate block
### {매장명}
- 위치:
- 예상 가격대:
- 인원 적합성:
- 공간 유형:
- 콜키지:
- 추천 이유:
- 유의사항:
- 근거 수준:
