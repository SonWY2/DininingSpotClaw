# 음식점 탐색 (Restaurant Explorer)

Claude Code를 활용한 회식 장소 탐색 자동화 도구입니다. 지역, 인원, 예산, 콜키지 등의 조건을 입력하면 웹 검색을 통해 최적의 식당 후보를 찾아 정리된 보고서를 생성합니다.

## 주요 기능

- **자연어 요청 처리**: "잠실 25명 회식 10만원" 같은 자연어로 식당 탐색
- **다각도 검색**: 지역/인원/예산/룸/대관/콜키지 기준으로 다양한 검색 쿼리 자동 생성
- **후보 검증**: 수용 인원, 가격, 콜키지 정보 등 플랫폼 정보 기반 검증
- **보고서 생성**: 비교 표와 상세 정보가 포함된 마크다운 보고서 출력

## 설치 방법

1. 저장소를 클론하거나 압축을 해제합니다.
2. 프로젝트 루트에 `.claude/` 폴더가 위치하도록 배치합니다.
3. Claude Code에서 해당 프로젝트 디렉토리를 엽니다.
4. `prompt.md`의 예시 프롬프트를 복사해 실행합니다.

## 사용 방법

```
잠실, 송파의 25명 이상 회식이 가능한 1인당 10만원 정도의 식당 찾아줘.
홀, 룸, 대관은 상관없고 콜키지 정보도 있으면 같이 찾아줘.
```

더 많은 예시는 [prompt.md](prompt.md)를 참고하세요.

## 프로젝트 구조

```
.claude/
├── agents/                          # 에이전트 정의
│   ├── dining-intake-normalizer.md  # 사용자 요구사항 정규화
│   ├── venue-source-researcher.md   # 웹 검색으로 후보 식당 수집
│   ├── venue-capacity-verifier.md   # 수용 인원 및 시설 검증
│   ├── venue-suitability-scorer.md  # 후보별 적합도 평가
│   └── venue-report-writer.md       # 최종 보고서 작성
└── skills/                          # 스킬 정의
    ├── dining-query-planner/        # 검색 쿼리 생성
    ├── dining-report-format/        # 보고서 포맷 규칙
    └── venue-verification-rules/    # 검증 기준
docs/                                # 생성된 보고서
prompt.md                            # 예시 프롬프트
install-guide.md                     # 설치 가이드
```

## 작동 원리

1. **요청 분석**: 사용자 입력을 지역/인원/예산/공간/콜키지 등으로 정규화
2. **검색 쿼리 생성**: 조건을 다양한 검색어 조합으로 확장
3. **후보 수집**: 웹 검색으로 식당 후보 탐색
4. **검증 및 평가**: 수용 인원, 가격, 편의사항 확인 및 점수 산정
5. **보고서 생성**: 비교 표와 상세 정보가 포함된 리포트 작성
